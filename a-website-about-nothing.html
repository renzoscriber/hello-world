Quantum mechanics in a nutshell

  1. States

    Every physical system has a state space which is a complex vector space. An isolated system's state can be described by a unit vector in that state space.

  2. Dynamics

    An isolated quantum system's evolution is described by a unitary matrix acting on the system's state space.

  3. Measurement = operators that extract information from a quantum system
  4. Combining systems = quantum state spaces combine via tensor products
